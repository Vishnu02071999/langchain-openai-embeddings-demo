
# OpenAI Embeddings with LangChain

This project demonstrates how to generate vector embeddings using OpenAI's text-embedding-3-large model through LangChain.

# Overview

Embeddings convert text into numerical vectors that capture semantic meaning. These vectors can be used for:

Semantic Search
Document Retrieval
Recommendation Systems
Text Similarity Analysis
Retrieval-Augmented Generation (RAG)

In this example, a sample sentence is converted into a 32-dimensional vector representation using OpenAI Embeddings.
